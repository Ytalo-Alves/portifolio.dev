# Portfólio — Ytalo Alves

Portfólio pessoal construído para comunicar meu posicionamento em `tech + operações`.

A proposta do projeto é apresentar minha trajetória entre logística, processos e tecnologia de forma clara, visual e objetiva, destacando atuação com sistemas, implantação de WMS/ERP, análise funcional e melhoria operacional.

## Contexto

- Perfil híbrido entre tecnologia e operações
- Foco em processos, sistemas e implantação
- Experiência prática com rotina operacional, documentação e suporte
- Portfólio pensado para mostrar clareza de raciocínio e visão de negócio

## Stack do projeto

- React + TypeScript + Vite
- Tailwind CSS
- Framer Motion
- React Router

## Seções

- `Início` com posicionamento e resumo profissional
- `Sobre` com trajetória, competências e diferenciais
- `Projetos` com seleção de entregas e detalhes
- `Serviços` com frentes de atuação em processos e sistemas
- `Contato` com formulário e canais diretos

## Funcionalidades

- Navegação por rotas
- Layout responsivo com foco em leitura rápida
- Cards, timelines e blocos de informação modular
- Contato via WhatsApp com mensagem formatada automaticamente
- Galeria de projetos com modal de detalhes

## Como rodar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Onde editar conteúdo

- Dados pessoais, serviços, projetos e links: `src/constants/index.ts`
- Páginas principais: `src/screens/`
- Componentes reutilizáveis: `src/components/`
- Navegação lateral e mobile: `src/components/layout/`

## Deploy

Compatível com Vercel, Netlify e outras plataformas que aceitam build padrão com saída em `dist/`.
